"Turtle" (Черепашка) - проект, позволяющий создавать различные геометрические фигуры с помощью команд управления курсорос(черепашкой).

Программа рисует линии (перемещение вперед и назад на заданное количество пикселей), позволяет управлять их направлением (повороты влево и вправо на заданный угол). 

Для пользования проектом у вас должен быть установлен компилятор C++. Вам необходимо перенести код в программу, скомпилировать ее, и в консоли вы сможете использовать весь функционал проекта с инструкциями для вводных данных.
